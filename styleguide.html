Portfolio Style Guide

This styleguide is meant to be used for the respective website in its design

Interactive Textual Combined

In-text link

CSS selector:

.intext

Sample Code:

<a href="#" class="intext">In-Text Link</a>

Rendered element:

Home Page

Headings

CSS selector:

.headingStyle

Sample Code:

<h1>Heading 1</h1> <h2>Heading 2</h2> <h3>Heading 3</h3>

Rendered element:

Heading 1

Heading 2

Heading 3
